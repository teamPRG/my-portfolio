# My Portfolio Website

A sleek website showcasing consulting and management services, property portfolio, and booking functionality.

## Features

- Responsive design using Bootstrap
- Services section
- Property portfolio gallery
- Calendly integration for booking appointments

## Setup

1. Clone or download the files.
2. Open `index.html` in a web browser to view the site.
3. To customize:
   - Edit `index.html` for content changes.
   - Modify `css/style.css` for styling.
   - Update `js/script.js` for JavaScript functionality.
4. Add property images to the `images/` folder and update the portfolio section in `index.html`.
5. Replace the Calendly URL in `index.html` with your actual Calendly scheduling link.

## Deployment

Host the files on any static site host like GitHub Pages, Netlify, or Vercel.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Calendly for scheduling
